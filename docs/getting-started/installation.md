# Installation

## Supported Versions

|         | Python 3.13 | Python 3.14 |
|---------|:-----------:|:-----------:|
| Django 5.2 | ✓ | ✓ |
| Django 6.0 | ✓ | ✓ |

## Install from PyPI

```bash
pip install django-prefetch-values
```

Or with uv:

```bash
uv add django-prefetch-values
```

## Development Installation

```bash
git clone https://github.com/oliverhaas/django-prefetch-values.git
cd django-prefetch-values
uv venv
uv sync --group dev
```
